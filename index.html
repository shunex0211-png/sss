📸 Instagram連携

Graph API Explorerからアクセストークンを取得して貼り付けます。

STEP 1 ― Graph API Explorerを開く

下のボタンでGraph API Explorerを開いてください。

STEP 2 ― トークンを生成
  1. 右上のドロップダウンで自分のアプリを選択
  2. 「アクセストークンを生成」をクリック
  3. 許可を求められたら「許可」
  4. 表示されたトークンをコピー
STEP 3 ― トークンを貼り付け

設定

Anthropic APIキー

① console.anthropic.com にログイン
② 左メニュー「API Keys」→「Create Key」
③ 発行されたキー（sk-ant-...）をコピーして上に貼り付け

Instagram API連携

Meta Developer → 設定 → ベーシック で確認できます。
App Secretを入力すると60日間有効な長期トークンに自動変換されます。
